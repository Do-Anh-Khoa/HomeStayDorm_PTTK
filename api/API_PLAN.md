# API Plan

Base URL local:

```text
http://localhost:5000/api
```

## Auth

| Method | Endpoint | Mục đích |
|---|---|---|
| POST | /auth/login | Đăng nhập |

## User

| Method | Endpoint | Mục đích |
|---|---|---|
| GET | /users | Danh sách người dùng |

## Room

| Method | Endpoint | Mục đích |
|---|---|---|
| GET | /rooms | Danh sách phòng/giường |

Sau này nhóm bổ sung API cho hồ sơ đăng ký, phiếu đặt cọc, hợp đồng, phiếu thu, dịch vụ, trả phòng.
